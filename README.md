# Automotive-CAD-Design-SolidWorks
SolidWorks-based automotive CAD design project featuring the modeling of 8 mechanical components and their final assembly.

<img width="1920" height="866" alt="spark plug" src="https://github.com/user-attachments/assets/52f867a0-1a6b-4782-833b-4e9cb6dbfef7" />


## Overview

This project presents a mechanical CAD design of an automotive assembly created using SolidWorks.

The project includes 8 individual mechanical parts and one final assembly.

## Software

* SolidWorks

## Project Contents

* 8 Part files (.SLDPRT)
* 1 Assembly file (.SLDASM)
* Assembly preview image

## Description

This project demonstrates 3D modeling and assembly design skills in mechanical engineering using SolidWorks.
